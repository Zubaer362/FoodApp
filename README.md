# FoodApp 

A modern food delivery web application inspired by Swiggy, built using React, Vite, and Tailwind CSS.

##  Features
- Restaurant listing & offers
- Responsive UI
- Search functionality
- Clean component-based architecture
- Fast performance with Vite

##  Tech Stack
- React
- Vite
- Tailwind CSS

##  Installation
```bash
npm install
npm run dev
```
